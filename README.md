# Kotlin-playground — 目的と学習指針

## 目的
このフォルダは、Kotlin の関数型プログラミングを理解するための学習用プレイグラウンドです。数学（素数、数列、アルゴリズムなど）に関する課題を並行して解くことで、プログラミング的思考と数学的センスを同時に養います。

## 学習対象
- Kotlin の関数型スタイル（高階関数、イミュータブルデータ、ラムダ、コレクション操作）
- 数学的トピック（素数判定、素因数分解、フィボナッチ、約数・倍数、モジュロ演算など）
- 問題のアルゴリズム化とテスト

## 学習フロー（小学生の自主学習形式）
1. めあて（学習前）を明確にする
2. 実装・演習を行う（短いコードスニペットで試す）
3. ふりかえり（学習後）を書く
4. 必要ならリファクタリング／追加課題に取り組む

## ファイル構成と命名規則（推奨）
- 各課題は `study-<YYYYMMDD>-<topic>.md` または `study-<topic>.md` のように保存
- 実装は `src/<topic>/` に Kotlin ファイルを置く（Gradle で管理する場合は `build.gradle` を追加）
- テンプレートは `templates/study-template.md` に置く

## めあて（学習前）
- 今日は、次のことができるようになる：
  - 例: 素数を判定する関数を関数型スタイルで書けるようにする

## 実装・メモ
- 使用言語: Kotlin
- 例:
```kotlin
fun isPrime(n: Int): Boolean =
    if (n < 2) false else (2..Math.sqrt(n.toDouble()).toInt()).none { n % it == 0 }
```

## ふりかえり（学習後）
- 出来たこと:
  - 例: 素数判定関数をラムダと `none` を使って書けた
- 難しかったこと / 改善点:
  - 例: 大きな数での効率化（試し割りの改良が必要）
- 次のめあて:
  - 例: 素因数分解を関数型で実装する
