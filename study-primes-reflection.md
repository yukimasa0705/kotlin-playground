- 素数判定で `n` の平方根まで調べれば十分なので、不要な繰り返しを減らすために使います。

```kotlin
fun isPrime(n: Int): Boolean =
    n >= 2 && (2..sqrt(n.toDouble()).toInt()).none { n % it == 0 }
```
- `isPrime` 関数は、与えられた整数 `n` が素数かどうかを判定します。
- `n >= 2` で 2 未満を素数ではないと判定します。
- `(2..sqrt(n.toDouble()).toInt())` で試し割りの範囲を `2` から `sqrt(n)` までに限定します。
- その範囲で `none { n % it == 0 }` が成り立つなら割り切れる数がなく、素数です。
- `&&` を使うことで、`n >= 2` が false なら後ろを評価せずに早期判定します。
